# 基金匹配器

> 没有最好的基金，只有最适合自己需求的 —— 一入向北

用数据帮你从 13,000+ 只基金里筛选出匹配度最高的。不追热点，不信排行榜，只看你的真实需求和量化数据。

## 两种版本

### 💻 终端版（电脑双击即用）
下载 `基金匹配器.exe`（Releases 页），双击运行。不需要装 Python。

### 🌐 网页版（手机/电脑都能用）
打开 https://你的链接.streamlit.app，直接在浏览器里用。

## 自己跑源码

```bash
pip install -r requirements.txt

# 终端版
python main.py

# 网页版
streamlit run app.py
```

## 怎么工作的

```
20 题引导式问答 → 四相淘汰算法 → 匹配度排名

相位1 (生存): 成立年限 · 经理从业 · 规模 · 申购状态 · 费率
相位2 (风险): 最大回撤 · 下行波动 · 回撤恢复速度
相位3 (效率): 主动型看索提诺+胜率 · 被动型看费率+跟踪
相位4 (偏好): 板块命中 · 风格偏好

每个子指标独立 ≥ 40% 才过关，不是加权平均混过去。
```
